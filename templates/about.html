About WyoBio Mobile BETA

WyoBio Mobile BETA

Citizen Science = Science for EverySone!

This mobile data collection application is currently in BETA form, and is intended to be used for testing purposes. The Wyoming Biodiversity Citizen Science Initiative (WyoBio) is a project intended to connect all people, especially Wyoming citizens, with information about biodiversity, including plants, animals and fungi.
